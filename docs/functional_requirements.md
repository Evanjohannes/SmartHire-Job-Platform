# Jobify - Functional Requirements

## User Management:

1. **US001 - Registration**

   - As a new user, I want to register as Job Seeker or Employer
   - Acceptance: Can create account with email, password, basic info

2. **US002 - Login**

   - As a registered user, I want to login to my account
   - Acceptance: Secure login, remember me option

3. **US003 - Profile Management**
   - As a user, I want to update my profile information
   - Acceptance: Can upload photo, edit bio, add skills

## Job Seeker Features:

4. **US004 - Job Search**

   - As a Job Seeker, I want to search for jobs by keyword, location, salary
   - Acceptance: Filters work, results load quickly

5. **US005 - Apply for Job**
   - As a Job Seeker, I want to apply for a job with one click
   - Acceptance: Application tracked, CV uploaded if needed
